Aufgabe 1d)

Erstellen Sie fr die beiden Geschlechter jeweils ein Stapeldiagramm bezogen auf die relativen Hufigkeiten des Merkmals Note. Welche Note war jeweils die schlechteste unter den besten 30%. Welche Note war jeweils die beste unter den schlechtesten 10%?

df %>%
  group_by(Geschlecht, Noten) %>%
  summarise("Absolute Hufigkeit" = n()) %>%
  mutate("relative Hufigkeit" = `Absolute Hufigkeit`/sum(`Absolute Hufigkeit`),
         Noten = factor(Noten)) %>%
  ggplot(aes(x = Geschlecht, y = `relative Hufigkeit`, fill = Noten)) +
  geom_col(width = .75) +
  scale_fill_brewer(palette = "Oranges") +
  geom_hline(yintercept = 0.7, color = "black") +
  geom_hline(yintercept = 0.1, color = "black") +
  theme_bw()
## `summarise()` has grouped output by 'Geschlecht'. You can override using the
## `.groups` argument.

Bei Frauen und Mnnern ist die schlechteste Note unter den besten 30% eine 2. Bei Frauen ist die beste Note unter den schlechtesten 10% eine 3. Bei Mnnern ist die beste Note unter den schlechtesten 10% eine 4.
